# App Store Promotional Text (170 character limit)

App Store Connect → My Apps → DealGapIQ → [Version] → iOS App → **Promotional Text**

Promotional text appears at the **very top of the App Description**, above the "...more" cut. Most users only see the first ~3 lines of description on mobile, so this is the only description text many will read. **It can be updated without submitting a new app version** — use this for time-sensitive offers, market commentary, or A/B testing.

---

## Recommended (PRIMARY)

> **See every US listing through an investor's lens. Pre-scored DEAL, MAYBE, or PASS. MLS + foreclosure + pre-foreclosure + auction. Discovery in seconds.** _(154 chars)_

**Why this wins**
- Opens with the hero headline metaphor ("investor's lens") — same language as the #1 screenshot, reinforcing message cohesion
- "Pre-scored DEAL, MAYBE, or PASS" introduces the Discovery framework that screenshots #3 and #4 then visualize
- Lists all four data sources explicitly — answers the most common qualifying question before users have to ask
- "Discovery in seconds" closes with the speed promise
- Front-loads the differentiated value (lens + pre-scoring) before the table-stakes feature list

## Alternates

### Alt 1 — Action-led _(160 chars)_
> **Stop scrolling listings. Every US property — MLS, foreclosure, pre-foreclosure, auction — comes pre-scored DEAL, MAYBE, or PASS. Hunt smarter, not harder.**

Strong if your install funnel data shows users come in already frustrated with Zillow/Redfin. Opens with a problem statement.

### Alt 2 — Feature-comprehensive _(160 chars)_
> **DealGapIQ scores every US listing for profit before you tap. Search MLS + foreclosures + auctions. See Target Buy, Income Value, Market Price — instantly.**

Strong if you want to surface specific financial outputs in the first impression. Names the three core Discovery metrics.

### Alt 3 — Time-sensitive (rotate during launch / promotions) _(168 chars)_
> **NEW: Off-MLS coverage now live. Foreclosure, pre-foreclosure, and auction listings — pre-scored DEAL, MAYBE, or PASS alongside MLS. Find what other apps miss.**

Use during the first 60 days post-launch to spotlight the differentiating feature. Replace once the "NEW" framing goes stale.

---

## Promotional Text Best Practices

- **Refresh every 30-60 days.** Promotional text is one of the few elements you can change without re-review. Stale promo text signals an inactive product.
- **Lead with differentiation, not table stakes.** Don't waste the first sentence on "search for properties" — every real estate app does that. Lead with what only DealGapIQ does.
- **Mirror the screenshot language.** If users see "Investor's Lens" on the hero screenshot and "Investor's Lens" again in promo text, the brand feels coherent and intentional.
- **Don't repeat the subtitle.** The subtitle and promo text appear back-to-back. Use them to layer information, not duplicate it.
